Selamat datang di FTIK Universitas Semarang Playground, tempat dimana kamu bisa main game keren berbasis web. Tampilan web ini menggunakan Big Picture a responsive site template buatan HTML5 UP, dibangun dengan Skel, dan dirilis secara bebas berdasarkan Creative Commons Attribution 3.0 license.

Ini Apa?

Game tidak selalu rumit, terkadang hanya cukup dibangun dengan bahasa dan teknologi yang sederhana. Disini tempatnya kamu bisa main game berbasis Web yang diambil dari Github.
Punya referensi game keren yang bisa ditambahkan disini? Silakan hubungi kami.
